# @wpdev/html-utils

DOM helpers and `elementProps()` for hydrating Preact/React widgets.

## Install

```bash
npm install @wpdev/html-utils
```

## Usage

```js
import { elementProps } from "@wpdev/html-utils";

const props = elementProps(container);
```
