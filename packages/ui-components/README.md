# @wpdev/ui-components

Shared Preact UI components (WDForm and related widgets) for wp-starter-kit consumers.

## Install

```bash
npm install @wpdev/ui-components
```

## Usage

```tsx
import { WDForm, createWDFormStore } from "@wpdev/ui-components";

const store = createWDFormStore({ fields: [] });
```

Subpath import:

```tsx
import { WDForm } from "@wpdev/ui-components/WDForm";
```
